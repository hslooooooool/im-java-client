# 消息服务SDK Java版本
IM消息服务Java SDK与Demo代码
## SDK说明
im-sdk-java项目为SDK源码。

如果使用IDEA，则将此module加入artifact->build后即可生成项目用SDK jar包。

## Demo说明
Demo采用Kotlin 1.3.x编写。工具为IntelliJ IDEA 2019.3。
